matrices
========

matrices is a pure-python matrix processing package with no C dependencies.

Example
-------

  >>> import matrices
  >>> matrix = matrices.Matrix([1, 2], [3, 4], [5, 6])
  >>> matrix
  <Matrix (3Ã—2)>
  >>> matrix.size()
  (3, 2)
